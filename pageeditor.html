Website Style & Navigation

Technical Description

As Student 2, my primary responsibility was to establish a consistent visual identity for the entire website. This involved creating a shared template, a global CSS file, and a responsive navigation bar.

  • Template Structure: I designed a common HTML template (`template.html`) with a `

  • Global CSS: A single external stylesheet, style.css, was created to define global styles. This includes the color palette (primary blue #0a73b7), typography (Inter font from Google Fonts), button styles, and footer design. Using a global CSS file ensures that changes can be made in one place and reflected across the entire site, adhering to the DRY (Don't Repeat Yourself) principle.
  • Navigation Bar: The navigation bar is fixed to the top using position: fixed;, so it remains accessible while the user scrolls. A hover effect was added to links for better interactivity, and a class .active will be used to highlight the current page. The navigation is fully responsive, adapting to different screen sizes.

Challenges and Lessons Learned

A key challenge was deciding on a style that all team members agreed upon and that fit the theme of UN Sustainable Development Goals. We overcame this through collaborative design sessions. I learned the importance of creating a scalable and maintainable CSS architecture from the start.

    Accessibility & Compliance

    Accessibility

    Throughout the development of my assigned pages, I focused on ensuring they were accessible. Key measures include:

    • Using semantic HTML to give structure and meaning to the content.
    • Providing `alt` text for all images to describe their content to screen readers.
    • Ensuring high contrast between text and background colors for readability.
    • Making interactive elements like links and buttons focusable for keyboard navigation.
